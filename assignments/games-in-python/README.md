
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input.

## 📝 Tasks

### 🛠️	Implement the Hangman Game

#### Description
Create a Python program that simulates the Hangman game. Players will guess letters to reveal a hidden word and try to complete the word before running out of attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when the word is guessed or attempts are exhausted
- Display clear win or lose messages

**Skills practiced:** String manipulation, loops, conditionals, random selection
